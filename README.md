# Cascade International School Website

Static multi-page website for Cascade International School, Bahadrabad, Haridwar.

## Pages

- Home
- About
- Academics
- Facilities
- Gallery
- Admissions
- Contact

## Features

- Responsive design for mobile, tablet, and desktop
- Sticky navigation with mobile menu
- Admission enquiry form
- Contact form
- Gallery with category filters
- Local SVG visual assets
- GitHub Pages friendly structure

## How To Open

Open `index.html` in any browser.

## GitHub Pages Deployment

1. Upload all files in this folder to the GitHub repository.
2. Open repository settings.
3. Go to `Pages`.
4. Set source to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

The site will be available at:

```text
https://Rishabh2619.github.io/cascade-school-website/
```
